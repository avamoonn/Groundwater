Welcome

WTR (Water Table Recharge) is a simple one-dimensional analytical groundwater model that you can interact with online to explore basic concepts of groundwater models and model non-uniqueness.

To navigate the WTR tool you can use menu items at the top to:

  • Learn about the underlying theory
  • Watch a video of an example application
  • Watch a video and/or read through the web page to learn how to use the model
  • Work through an educational exercise
  • explore additional material, or
  • run the WTR tool from the Green Run WTR button
