Welcome

DFPS (Pumping Well Depletion of a Fully Penetrating Stream) is an analytical groundwater model that you can interact with online to explore basic concepts of the impact of pumping a well near a stream.

To navigate the DFPS tool you can use menu items at the top to:

  • Learn about the underlying theory
  • Watch a video and/or read through the web page to learn how to use the model
  • Run the DFPS tool from the green "Run DFPS" button
